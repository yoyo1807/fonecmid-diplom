> **Инструкция по запуску автоматизированного теста**

 **1. Подготовка информационной базы:**
 Для корректного прохождения сценария база должна быть подготовлена следующим образом:
*   **Пользователи:** В информационной базе должны быть созданы три пользователя со следующими именами (логинами):
    1. `Менеджер` (права: "БазовыеПрава", "Менеджер (ВКМ)")
    2. `Специалист`(права: "БазовыеПрава", "Специалист (ВКМ)")
    3. `Бухгалтер` (права: "БазовыеПрава", "Бухгалтер (ВКМ)")
    *Всем пользователям необходимо назначить полные права (или роли, соответствующие их должностям).*

*   **Режим запуска:** Основной сеанс 1С (Тест-менеджер), в котором открывается Vanessa Automation, должен быть запущен с параметром: `/testmanager`.

#### 2. Настройка окружения (Vanessa Automation)
Поскольку сценарий предполагает переключение между ролями, необходимо настроить **Тест-клиенты** в интерфейсе Vanessa Automation:
1.  Запустите Vanessa Automation в режиме «Предприятие».
2.  Перейдите на вкладку **Клиенты теестирования**.
3.  Добавьте три записи, чтобы сценарий мог обращаться к ним по именам:
    *   Имя: `Менеджер`, Пользователь: `Менеджер`.
    *   Имя: `Специалист`, Пользователь: `Специалист`.
    *   Имя: `Бухгалтер`, Пользователь: `Бухгалтер`.
4.  В колонке «Путь к базе» или «Строка подключения» укажите путь к текущей информационной базе.
5.  Убедитесь, что на вкладке **Основные** выбран тип клиента (например, «Тонкий клиент»).

#### 3. Запуск сценария
1.  Нажмите кнопку «Загрузить feature-файл» и выберите файл сценария.
2.  Нажмите «Запустить» (F5).
3.  Vanessa Automation поочередно будет запускать новые окна 1С от имени нужных пользователей и выполнять шаги по планированию, закрытию обслуживаний и созданию актов.
